## 🚀 Overview  
The **Global IPI Platform** is a full-stack web application designed to monitor, analyze, and manage global intellectual property (IP) data.  

It enables innovators, law firms, and R&D teams to track patents, trademarks, and IP filings while gaining insights through advanced analytics and AI-driven features. :contentReference[oaicite:0]{index=0}  

The platform evolved from a basic authentication system into a complete SaaS-based IP intelligence solution with search, analytics, filing management, and admin capabilities.

---

## ✨ Key Features  

### 🔐 Authentication System  
- Login & Register pages with validation  
- Secure routing using React Router  
- Responsive UI across devices  

---

### 🔍 Advanced IP Search  
- Search patents & trademarks using:  
  - Keyword  
  - Assignee  
  - Inventor  
  - Jurisdiction  
  - Status  
- Filtering, sorting, and pagination  
- Toggle between Patent & Trademark search :contentReference[oaicite:1]{index=1}  

---

### 📊 Data Visualization & Analytics  
- Interactive charts using Recharts  
- Search results in card/list view  
- Real-time trend insights  

---

### 🌍 Global Map Integration  
- Google Maps API integration  
- Region-wise IP distribution  
- Clickable markers with details  
- Map synced with search results :contentReference[oaicite:2]{index=2}  

---

### 💬 Feedback & Contact System  
- Firebase Firestore integration  
- Feedback form & contact form  
- User interaction tracking

### 💼 SaaS Subscription System  
- Free, Pro, and Enterprise plans  
- Feature-based access control  
- Mock payment system

- Track IP lifecycle (Filed → Granted → Expiry)  
- Filing dashboard with analytics
